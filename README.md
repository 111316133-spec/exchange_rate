
# 📘 台灣匯率即時監控系統 - 使用說明書

## 1. 系統簡介

本系統是一個網頁版的匯率監控儀表板，具備 **每秒自動更新**、**雙向匯率換算** 以及 **歷史走勢圖表** 功能。無需安裝任何軟體，直接使用瀏覽器（Chrome、Edge、Safari）開啟 HTML 檔案即可使用。

> **⚠️ 重要提醒**：
> 本系統目前顯示的數據為 **「模擬演示數據」**（數值會隨機波動），僅供功能測試與介面展示使用，**請勿用於真實金融交易參考**。

---

## 2. 功能詳細操作

### ❶ 即時匯率看板 (Dashboard)

位於畫面上方，顯示各國貨幣對台幣的參考匯率。

* **自動更新**：每秒鐘更新一次數值。
* **漲跌提示**：
* 🔴 **紅色 (▲)**：代表匯率上漲（相較於上一秒）。
* 🟢 **綠色 (▼)**：代表匯率下跌。


* **支援幣別**：目前顯示 USD (美金)、EUR (歐元)、CNY (人民幣)、KRW (韓元)、AUD (澳幣)。

### ❷ 匯率計算機 (Converter)

位於畫面中間，支援台幣與各國貨幣的雙向換算。

**操作步驟：**

1. **輸入金額**：在左側格子輸入您想換算的數字（例如 `1000`）。
2. **選擇持有貨幣**：在第一個下拉選單選擇您手上的幣別（例如 `TWD`）。
3. **選擇目標貨幣**：在第二個下拉選單選擇您想兌換的幣別（例如 `USD`）。
4. **查看結果**：下方會即時顯示換算後的金額，以及當下的參考匯率。

**🌟 進階功能：快速交換 (Swap)**

* 點擊中間橘色的 **「⇄」按鈕**。
* 系統會自動將「持有貨幣」與「目標貨幣」對調，方便您快速查看反向匯率（例如從「台幣換美金」變成「美金換台幣」）。

### ❸ 歷史走勢與查詢 (Charts)

位於畫面下方，提供視覺化的匯率波動圖。

**操作步驟：**

1. **切換幣別**：在圖表標題旁的下拉選單，選擇想查看的貨幣（例如從 USD 切換到 EUR）。
2. **切換區間**：
* 點擊 **「近 7 天」**：查看短期波動。
* 點擊 **「近 1 個月」**：查看較長期的趨勢。


3. **查詢特定日期**：
* 點擊日曆圖示選擇特定日期。
* 按下 **「查詢特定日」** 按鈕。
* 系統會跳出視窗告知該日期的（模擬）平均匯率。

## 3. 常見問題 (Q&A)

**Q: 為什麼日圓 (JPY) 不見了？**
A: 根據您的需求，日圓已從監控列表與換算選項中移除。

**Q: 為什麼我重新整理網頁後，圖表就不一樣了？**
A: 因為本系統使用的是「模擬亂數產生器」，每次重新整理網頁（F5），系統會重新生成一組全新的隨機歷史數據。

**Q: 如何修改預設的匯率基準？**
A: 若您懂一點程式碼，可以用記事本開啟 HTML 檔，搜尋 `const currencies`，修改裡面的 `baseRate` 數值即可。例如將美金從 `31.5` 改為 `32.0`。
