# 📘 Global FX Pro 即時匯率監控系統 - 使用說明書

歡迎使用 **Global FX Pro**！這是一個結合「真實國際匯率基準」與「秒級動態模擬」的視覺化匯率看板。本系統旨在模擬金融市場的瞬息萬變，提供使用者流暢、現代化的匯率監控體驗。

---

## 1. 系統介面概覽

當您開啟網頁時，系統會自動連線至國際匯率 API 取得今日基準價。介面主要分為三個區塊：

1. **頂部資訊列**：顯示系統名稱與系統時間（Live），確認連線狀態正常。
2. **匯率儀表板 (Dashboard)**：顯示主要貨幣對新台幣 (TWD) 的即時報價。
3. **智能換算 (Calculator)**：支援即時連動的貨幣計算機。
4. **市場趨勢監控 (Chart)**：可視化的匯率走勢圖。

## 2. 功能詳細介紹

### 📊 A. 即時匯率儀表板

位於畫面上方的卡片區塊，監控五大主流貨幣（美金、歐元、人民幣、韓元、澳幣）。

* **報價更新**：每一秒鐘，數值會根據模擬的市場波動進行更新。
* **漲跌顏色指標**（採用台灣股市慣例）：
* 🔴 **紅色 (▲)**：代表匯率**上漲** (升值)。
* 🟢 **綠色 (▼)**：代表匯率**下跌** (貶值)。


* **膠囊標籤**：卡片下方的膠囊顯示自您進入網頁以來的漲跌幅差額。

### 🧮 B. 智能換算計算機

這不是一般的靜態計算機，它與市場波動完全同步。

1. **輸入金額**：在「持有金額」欄位輸入數字。
2. **選擇幣別**：透過下拉選單選擇「持有幣別」與「換算目標」（選單已包含中文名稱，如：USD 美金）。
3. **雙向切換**：點擊中間的 **「⇄ 交換按鈕」**，可快速互換持有幣別與目標幣別。
4. **動態結果**：
* 即使您停止輸入，只要市場匯率跳動，**換算結果也會跟著每一秒自動更新**。
* 下方小字會顯示當前的精確匯率（例如：`1 USD ≈ 32.4501 TWD`）。



### 📈 C. 市場趨勢監控圖表

位於頁面最下方的折線圖，提供歷史與即時走勢的視覺化分析。

1. **切換幣別**：點擊圖表右上角的下拉選單（例如選擇 `USD 美金 / TWD`），圖表將立即切換。
2. **漸層視覺**：圖表採用現代化的藍色漸層填充，清晰易讀。
3. **Live 動態尾端**：
* 圖表左側為過去 30 天的模擬歷史走勢。
* **請注意圖表的最右端**：線條的尾巴會隨著每一秒的匯率波動而上下「抖動」，真實呈現市場的生命力。

## 3. 常見問題 (Q&A)

**Q: 為什麼我沒有按重新整理，數字卻一直在變？**
A: 本系統內建「即時連動引擎」，每秒鐘會自動模擬市場微幅波動並更新畫面上所有數據（包含計算機結果與圖表），無需手動重新整理。

**Q: 這些匯率是真實的嗎？**
A:

* **基準價**：來自真實的國際匯率 API (`ExchangeRate-API`)，確保大方向準確。
* **秒級波動**：為了展示動態效果，秒級的微幅跳動是由程式演算法模擬生成的（基於真實價格的 ±0.06% 浮動），僅供觀察體驗，**請勿作為實際高頻交易依據**。

**Q: 支援手機瀏覽嗎？**
A: 是的。系統採用響應式設計 (Responsive Design)，在手機、平板或電腦上都能完美顯示。
